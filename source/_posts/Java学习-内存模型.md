### 3.1事前准备

注意区别于内存区域，内存区域指的是JVM运行时将数据分区域存储，是对内存空间的划分。

Java内存模型(Java Memory Model)，是共享内存的并发模型，线程之间主要通过读-写共享变量（堆内存中的实例域，静态域和数组元素）来完成隐式通信。

Java 内存模型（JMM）控制 Java 线程之间的通信，决定一个线程对共享变量的写入何时对另一个线程可见。
